Open Any File Data Collection Notice

SUMMARY ---------------------------------------------------------------------------------

Open Any Files is a universal file opener for Windows that allows you to open a wide variety of file types and view the contents of them. In order to develop and improve product and service experience, Open Any File collects and analyzes user behaviors information inside the app. All these information is not personal identifiable according to the Privacy Policy.

 

Note: If you do not want to allow Open Any File to collect any personal data, do not install
